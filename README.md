# Gym Management System

A DBMS project developed for managing gym operations such as member records, trainers, attendance, payments, and health metrics using SQL and relational database concepts.

## Features

- Member Management
- Trainer Management
- Attendance Tracking
- Membership Plan Management
- Payment Management
- Health Metrics Tracking
- BMI Calculation
- Stored Procedures and Triggers

## DBMS Concepts Used

- ER Modeling
- Normalization (up to BCNF)
- Functional Dependencies
- Relational Schema Design
- SQL Queries
- Triggers
- Stored Procedures
- Functions

## Technologies Used

- SQL
- MySQL / MariaDB
- DBMS Concepts

## Database Tables

- Members
- Trainers
- MembershipPlans
- Payments
- Attendance
- HealthMetrics
